# BleacherStream
Bleacher Stream is an app that will to be a game stream for little league baseball games. It works in conjuction with the Bleacher Stream Server repository.

## Building

The app can be built (and run) on PC but the intention is to run on Android and IOS (IOS functionality has not been tested and cannot be guranteed).

Qt Creator is recomended but not required. Make sure any SDK and build tools are correctly installed.v Additionally, Qt will need to be installed along with all Android or IOS tools. Building for IOS may only be available on MacOS. Finally, run CMake with the correct configuration.
